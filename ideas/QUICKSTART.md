# 🚀 Quick Start Guide - Comptable AI

## Start the Application

1. **Open Terminal** in the `comptable-ai` folder

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open your browser** and go to:
   ```
   http://localhost:3000
   ```

That's it! 🎉

## What You'll See

- A beautiful welcome screen with the Comptable AI logo
- Three example questions you can click to test
- A chat input at the bottom supporting French and Arabic
- Professional Moroccan-inspired design (green and gold colors)

## Testing the Chat

1. Click one of the example questions, OR
2. Type your own question in French or Arabic
3. Press Enter or click the send button
4. Watch the AI respond with a beautiful animation

## Example Questions (Click to Try)

- Comment calculer la TVA au Maroc ?
- Quelles sont les obligations fiscales d'une SARL ?
- Quelle est la différence entre l'IS et l'IR ?

## Features to Notice

✅ **Auto RTL Detection** - Type in Arabic and see it automatically align right  
✅ **Smooth Animations** - Messages slide in beautifully  
✅ **Loading Indicator** - Bouncing dots while waiting for response  
✅ **Responsive Design** - Try resizing your browser window  
✅ **Professional Look** - Looks like a real product  

## Keyboard Shortcuts

- **Enter** - Send message
- **Shift + Enter** - New line in message

## Stop the Server

Press `Ctrl + C` in the terminal to stop the development server.
